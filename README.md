## ✨ 功能特性

### ⏱️ 时间调节
- **点击选择** - 点击时/分/秒选中对应部分
- **滚轮调节** - 鼠标滚轮上下调节数值
- **进度圆环** - 实时显示当前值的进度
  - 0 → 空圆环
  - 最大值 → 完整圆环
- **颜色区分** - 小时(金色)、分钟(蓝色)、秒(粉色)

| 单位 | 范围 | 颜色 |
|------|------|------|
| 小时 | 0-24 | 🟡 金色 |
| 分钟 | 0-59 | 🔵 蓝色 |
| 秒   | 0-59 | 🩷 粉色 |

## 🚀 安装运行

### 方式一：直接运行 exe（推荐）

**OneDir 版本：**
1. 将 `dist/Pomodoro_Dir/` 整个文件夹复制到任意位置
2. 双击 `Pomodoro_Dir.exe` 即可运行

**OneFile 版本：**
1. 将 `dist/Pomodoro_File.exe` 复制到任意位置
2. 双击 `Pomodoro_File.exe` 即可运行

### 方式二：从源码运行

**安装依赖：**
```bash
pip install customtkinter pygame numpy scipy Pillow
```

或使用 conda 环境：
```bash
conda create -n Pomodoro python=3.12
conda activate Pomodoro
pip install customtkinter pygame numpy scipy Pillow
```

**运行程序：**
```bash
cd "c:/Users/mc/Desktop/practice  code/pomodoro-timer"
python main.py
```

## ⌨️ 操作说明

### 主界面按钮

| 按钮 | 功能 |
|------|------|
| ▶ 开始 | 启动计时 |
| ⏸ 暂停 | 暂停计时 |
| 🔄 重置 | 重置当前计时 |
| ⏭ 跳过 | 跳过当前阶段（不卡顿） |
| ✨ 特效 | 切换7种粒子效果 |
| 🎵 音乐 | 打开白噪音混音器 |
| 🎨 主题 | 切换5种主题 |
| ⚙️ 设置 | 打开设置面板 |

### 时间调节

1. **点击** `00` 选中小时 → 滚轮调节 (0-24)
2. **点击** `25` 选中分钟 → 滚轮调节 (0-59)
3. **点击** `00` 选中秒 → 滚轮调节 (0-59)

### 音乐混音器

1. 点击 **🎵 音乐** 打开混音器
2. 点击任意白噪音按钮**开关**
3. 可以**同时开启多个**，混合出你喜欢的环境音
4. 使用**主音量**滑块调节整体音量
5. 点击**全部停止**一键关闭所有音轨

## 💡 番茄工作法

番茄工作法是一种时间管理方法：

1. **选择一个任务**
2. **设定25分钟计时**
3. **专注工作直到计时结束**
4. **短暂休息5分钟**
5. **每4个番茄后，长休息15-30分钟**

### 好处

- ✅ 提高专注力
- ✅ 减少拖延
- ✅ 平衡工作与休息
- ✅ 提高生产力

## 🐛 常见问题

### Q: 点击按钮时会闪烁吗？
A: 不会，已使用 `transient()` 确保弹窗在主窗口上方，且不会导致闪烁。

### Q: 可以同时播放多个白噪音吗？
A: 可以！支持多轨混合播放，同时开启多个音轨。

### Q: 跳过时会卡顿吗？
A: 不会，跳过操作已优化为非阻塞式。

### Q: 如何添加自己的白噪音？
A: 将 MP3/WAV 文件放到 `sounds` 文件夹，然后在音乐面板选择 **📁 选择音乐文件**。

### Q: 粒子特效会影响性能吗？
A: 已优化为 20fps，粒子数量限制在 500 个，不会影响性能。

### Q: exe 文件打不开怎么办？
A: 参考 `打包问题修复说明.md` 和 `OneFile打包修复说明.md` 文档。
